This packages allows to analyze output of log4j. The analyze consists of two steps. The first one is the initialization of the capturing process. This can be done via HTML using a {@link fitgoodies.log4j.SetupFixture} or via java using the {@link fitgoodies.log4j.LogHelper}. The second step is the analysis of the captured events. This is done with the {@link fitgoodies.log4j.LogFixture}.

@since 1.0.0
