Aliases enable a user to type less! After defining an alias, a fixture can be used by specifying its alias name instead of its fully qualified class name.

Aliases require a runner of {@link fitgoodies.runners}.

Example:

fitgoodies.alias.SetupFixture
myFixtureorg.example.fixture.MyOwnFixture

Now you can use:

myFixture
.........
.........

@since 1.0.0
