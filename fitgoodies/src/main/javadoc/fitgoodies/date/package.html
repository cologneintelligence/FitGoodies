This package contains helpers to parse Dates and Times. The {@link fitgoodies.date.SetupHelper} allows to set a locale and a format string, so that all parsers and adapters, which use dates, can be configured to "understand" the users input.

Use {@link fitgoodies.date.SetupFixture} can be used to configure the format in HTML input, the {@link fitgoodies.date.SetupHelper} can be used to setup the formats from java.

@since 1.0.0
