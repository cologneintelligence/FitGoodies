Readers are used to process files which contains any kind of record sets. A reader must be able to read a record and to retrieve the fields of this record. Readers are used by {@link fitgoodies.file.AbstractFileRecordReaderFixture}s to compare a file's content with a HTML table.

This package already contains reader for fixed length value files ( {@link fitgoodies.file.readers.FixedLengthRecordReader}), for example: 

this    is  lineone
and     thislinetwo
The field length are 8, 4, 4, 3.

It also contains two readers for delimited files. The simple {@link fitgoodies.file.readers.DelimiterRecordReader} process delimited records: 
this|is|line|one
and|this|line|two


The more sophisticated {@link fitgoodies.file.readers.CSVRecordReader} processes CSV files which may also use the delimiter char in their fields.

@since 1.0.0
