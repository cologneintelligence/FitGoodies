This package allows fit to convert Strings into Objects. Fit needs to convert your HTML table into objects to be able to compare a cell value with an object's value. This is done via a parse method. Fit can parse numbers and strings.

This package provides tools to extend the parse features. This makes it possible to add parsers for custom data types. A parser must implement {@link fitgoodies.parsers.Parser} and can be registered via java code using {@link fitgoodies.parsers.ParserHelper} or via HTML using {@link fitgoodies.parsers.SetupFixture}.
All Parsers in this namespace are registered by default.

If parsing is not enough and you want to provide your own toString or equals method, it is better to implement a {@link fitgoodies.adapters.AbstractTypeAdapter}.

@since 1.0.0
