This namespace provides tools for interacting with files. The {@link fitgoodies.file.FileFixture} can be used to select files via HTML. Its values can be received using the {@link fitgoodies.file.FileFixtureHelper}.

The fixtures {@link fitgoodies.file.CSVFileRecordFixture}, {@link fitgoodies.file.DelimiterFileRecordFixture} and {@link fitgoodies.file.FixedLengthFileRecordFixture} can check a file's content. The {@link fitgoodies.file.XMLFileFixture} can analyze a XML file using XPath expressions.

@since 1.0.0
